# Objective 2: Customer-Based Booking Interface

**Owner (Member Name):** Noilee Ann S. Pagkaliwagan  
**Objective Title:** Customer-Based Booking Interface  

## Objective Description
This objective centers on the development of a remote, self-service Progressive Web App (PWA) portal that features an online booking system, date-specific product availability checking, secure PayMongo downpayments, and automated Semaphore SMS notifications. It matters to the overall project because it modernizes the client experience, protecting customers from scheduling errors and removing the need for repeated manual inquiries or physical store visits. It integrates with Objective 1 by pulling real-time inventory data to prevent double-booking conflicts, while simultaneously feeding newly confirmed reservation and payment data into the database for Objective 3 to utilize in demand forecasting.
