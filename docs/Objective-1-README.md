# Objective 1: Rental Management System

**Owner (Member Name):** Dayao, Baby Joy C.  
**Objective Title:** Rental Management System  

## Objective Description
This objective covers the foundational digital infrastructure for Mylene's Boutique, focusing on the implementation of role-based access control, the full transaction management lifecycle, and real-time inventory monitoring. It is critical to the overall capstone because it replaces a disorganized, 23-year reliance on manual handwritten logbooks with a secure, centralized digital database, significantly reducing transaction retrieval times. Functioning as the core data engine, this module connects directly to the other objectives by supplying the synchronized inventory statuses necessary for the customer booking portal.
